1 # BTstack Arduino Manual
2 
3 Thanks for checking out BTstack on the Arduino platform!
4 
5 In this manual, we first provide a quick tutorial that helps install the BTstack Library for the Arduino IDE and gets you started with the examples.
6 
7 In the second part, we'll explain the examples to get you starting with creating your own Sketches that use the BTstack LE Arduino Shield.
8 
9 ## Supported Harware
10 
11 It should work with any board that provides at least 64 kB Flash and a working SPI driver.
12 So far, we did test the BTstack LE Arduino Shield together with these Arduino boards:
13 
14 - Arduino Zero
15 - Arduino Mega 2560 R3
16 - Arduino Mega 2560
17 - Arduino Mega 1280
18 - Sparkfun Mega 3.3
19 
20 Please let us know if you got it working with other Arduino clones.
