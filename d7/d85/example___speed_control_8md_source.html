1 # SpeedControlクラスの使い方
2 
3 SpeedControlクラスの仕様？については[こちら](https://korosuke613.github.io/etrobocon2017/dd/dc1/class_speed_control.html)をご覧ください。
4 
5 ## 概要
6 SpeedControlクラスはEV3をPID制御を用いて速度制御するためのクラスです。PIDクラスを継承しています。
7 
8 ## 使い方
9 ### 1. インクルードする
10 ```c++
11 #include "SpeedControl.h"
12 ```
13 
14 ### 2. コンストラクタを作成する
15 ```c++
16 SpeedControl sp;
17 ```
18 
19 ### 3. PIDパラメータを設定する
20 ```c++
21 sp.setPid(Kp, Ki, Kd, target);
22 ```
23 [setPidメソッド](https://korosuke613.github.io/etrobocon2017/dc/de1/class_pid.html#aa8626402086bcd10cbb7f581c92300fb)でPIDパラメータを設定できます。
24 * Kp - 比例定数
25 * Ki - 積分定数
26 * Kd - 微分定数
27 * target - 目標値
28 
29 ### 4. PID計算をする
30 ```c++
31 value = sp.calculateSpeedForPid(right_motor, left_motor);
32 ```
33 [calculateSpeedForPidメソッド](https://korosuke613.github.io/etrobocon2017/dd/dc1/class_speed_control.html#a4dc24488ce1566fe7b2dc1ecaf1a7bc8)でPID計算ができます。
34 * value - 実際に計算された結果
35 * right_motor - 右モータの回転角
36 * left_motor - 左モータの回転角
37 
38 **この計算は4msごとに実行してください。**
39 
40 ## 使用例
41 [ソースはこちら](https://gist.github.com/korosuke613/9b595f525988115c085337dc386d83e6)
42 
43 今回はPIDパラメータを以下のように設定しています。
44 * Kp = 2.0
45 * Ki = 2.0
46 * Kd = 0.024
47 * 目標値 = 70.0
48 
49 ```c++
50 #include <iostream>
51 #include <random>
52 #include <unistd.h>
53 /** インクルード */
54 #include "SpeedControl.h"
55 
56 // グローバル変数
57 int speed100ms, value;
58 int right_motor, left_motor;
59 
60 // プロトタイプ宣言
61 void calcVirtualWheels();
62 
63 int main(){
64  speed100ms = value = right_motor = left_motor = 0;
65 
66  /** １．コンストラクタを作成する */
67  SpeedControl sp;
68 
69  /** ２．PIDパラメータの設定する
70  * 今回は以下のようにしています
71  * Kp = 2.0
72  * Ki = 2.0
73  * Kd = 0.024
74  * 目標値 = 70.0
75  */
76  sp.setPid(2.0, 2.0, 0.024, 70.0);
77 
78  // 注）4msごとに実行されると仮定して2500回(10秒)回してます
79  for(int i=0; i<2500; i++){
80  speed100ms = sp.getSpeed100ms();
81  calcVirtualWheels();
82  /** ３．PID計算をする */
83  value = sp.calculateSpeedForPid(right_motor, left_motor);
84  // 結果を画面に出力
85  std::cout << "スピード(100ms) " << speed100ms << ", PID値 " << value << ", 前進値" << (right_motor + left_motor) / 2 << std::endl;
86  usleep(4000);
87  }
88  return 0;
89 }
90 
91 // 仮想的なモータエンコーダです。
92 void calcVirtualWheels(){
93  // モータの前進にランダム性を持たせる
94  std::random_device rnd;
95  std::mt19937 mt(rnd());
96  std::uniform_int_distribution<int> rand50(0,50); // 1~10の一様乱数
97  double r_value = rand50(mt) * 0.1;
98 
99  // 実際に進んだ距離とする
100  left_motor = right_motor += (speed100ms/25 + value + r_value) * 0.1;
101 }
102 ```
103 ### コンパイル方法
104 ```bash
105 g++ -w ../src/Pid.cpp ../src/SpeedControl.cpp example_SpeedControl.cpp -I../include
106 ```
